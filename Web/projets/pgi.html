Création d'un module de statistique

Ceci est le deuxième gros projet de ma deuxième année de BUT Informatique, où l'on devait réaliser un module.

Ce projet a été réalisé en groupe de 3 lors de ma deuxième année de BUT Informatique. Il fait suite au premier projet de cette même année. Nous devions après avoir créer nos jeux, les assembler sous un launcher commun et intégrer de nouveaux modules. Notre groupe a décidé de créer un module de statistique réservé aux admins.

Pour la création de ce module, nous avons utilisé Python pour traiter les statistiques, SQL pour créer des fonctions afin de récupérer certaines données, et enfin une surcouche Java pour exécuter le Python ainsi que du JavaFX.

Ce module de statistique permet aux administrateurs de la borne d'arcade d'accéder à des données précises sur leur borne, comme par exemple le nombre de joueurs actifs, le nombre de tournois créés ou encore le jeu le plus joué.

Grâce à ce module, les administrateurs pourront prendre des décisions selon les données et statistiques données concernant la gestion de leur borne d'arcade et de leurs jeux.

Pour ma part, j'ai notamment réalisé plusieurs fonctions SQL pour récupérer des données précises, et je me suis occupé d'une très grande partie du Python afin de récupérer ses données et les transformer en réelle statistiques, par exemple en prenant chaque jours/semaines des données pour les transformer en graphique.
